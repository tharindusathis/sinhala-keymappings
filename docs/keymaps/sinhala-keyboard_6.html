Key:
1st Row - QWERTY layout
2nd Row - long press
3th Row - short press

al - al-lakuna
rep - repaya
rak - rakaaranshaya
yan - yansaya
m-lu - muurdhaja lu
JOIN - join previous letter with the next
TOUCH - touch previous letter with the next
ZWJ - zero width joiner
ZWNJ - zero width non-joiner

Objective

Create a virtual keyboard layout where there are two types of key presses, short and long. A short key press will insert the symbol on the key. A long key press will pop up a dynamic list of keys to select. The dynamic list of keys that pop up will contain symbols that are phonetically related to the symbol on the key that was long-pressed.

Place Sinhala letters only on keys corresponding to the letters of the Latin alphabet. Place Sinhala punctuation only on keys corresponding to Latin punctuation.
